# Get started with Crestone

Crestone is an innovative data integration software that helps companies synchronize, transform, and transfer data between different business systems, such as ERP, CRM, databases, and analytics platforms.

It is designed to provide an intuitive experience for managing data flows, reducing complexity and saving time in integration projects.

## Goals and scope

- **Main Objective:** Enable seamless connections between data sources and destinations in a simple and intuitive way.
- **Scope**: Crestone is primarily aimed at technical users (e.g., IT administrators, data engineers) who have the necessary access credentials to configure and manage the integrations.z
