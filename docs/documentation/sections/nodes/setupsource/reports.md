## Create a Report in Extraction Node 
### 1. Selecting an SAP ABAP system
![SAP ABAP system selection screen](/img/node/reports/a.png)

### 2. Select extraction type "Report".
![Report extraction type selection](/img/node/reports/b.jpeg)

### 3. Select the type of transaction
![Transaction type selection interface](/img/node/reports/c.jpeg)

### 4. Select a transaction code
![Transaction code selection screen](/img/node/reports/d.jpeg)

### 5. Select variant if necessary
![Variant selection interface](/img/node/reports/f.jpeg)

### 6. Go to Report Settings to detect the columns
![Report Settings navigation for column detection](/img/node/reports/g.jpeg)


### 7. Click "Automatically detect columns" to set the width and offset of each column.
![Automatic column detection button](/img/node/reports/h.jpeg)

### 8. Establish the rows to be omitted if necessary
![Row omission configuration screen](/img/node/reports/i.jpeg)

### 9. Adding a pattern to skip rows by keyword
![Pattern configuration for keyword-based row skipping](/img/node/reports/j.jpeg)

### 10. Click on the confirmation button.
![Confirmation button location](/img/node/reports/k.jpeg)

### 11. Go to Preview to verify that the result is as expected.
![Preview screen showing extraction results for verification](/img/node/reports/l.jpeg)
